JSON vs API - Difference in Hinglish

1. JSON Kya Hai?

JSON (JavaScript Object Notation) ek lightweight data format hai jo data ko store aur exchange karne ke liye use hota hai.

Example:

{
  "name": "Meghraj",
  "age": 25,
  "city": "Pune"
}

2. API Kya Hai?
