Event Basics in JavaScript

JavaScript mein events ko handle karna ek basic concept hai. Jab hum kisi element ke saath koi action perform karte hain (jaise click, hover, keypress), toh ek event trigger hota hai. Aaj hum basic events ko samjhenge.




Explanation

1. Jab button ko click karte hain, tab ek "click" event trigger hota hai aur ek alert box show hota hai.
2. Jab input field mein type karte hain, toh "input" event trigger hota hai aur console mein ek message print hota hai.
